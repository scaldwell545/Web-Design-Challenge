Cloudiness

Unsurprisingly, there doesn't seem to be any correlation between cloudiness and latitude. The data does illustrate, however, that most entries were around the same values (i.e. rounded numbers). This trend can be seen by the lines that can be drawn horizontally across the graph, and this might indicate that there is an issue with accuracy for the metric to determine cloudiness.
